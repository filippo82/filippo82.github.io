How to set up python on your computer

September 5, 2022 One-minute read
development

Howdy folks!

I recommend pyenv

pyenv Link to heading

To install it, we use brew:

brew update
brew install pyenv pyenv-virtualenv

Then, we append these lines at the end of the ~/.zshrc file:

# pyenv config
export PYENV_ROOT="$HOME/.pyenv"
command -v pyenv >/dev/null || export PATH="$PYENV_ROOT/bin:$PATH"
eval "$(pyenv init -)"
eval "$(pyenv virtualenv-init -)"

pyenv-virtualenv

First, we list all Python versions available to pyenv:

$ pyenv versions
